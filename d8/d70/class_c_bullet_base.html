説明

飛道具基本クラス

CGoluahObjectを飛び道具向けに拡張したクラスです。 act_run() : 飛んでいるときのアクション　、　 act_bomb() : ヒット後、消滅するまでのアクション　、　 act_run2() : ヒット後も消滅せずに(攻撃力を失って)飛んでいくアクション　、　 を実装することにより簡単に飛び道具クラスが作れます。

攻撃力は通常、atkinfoメンバ変数が使用されます。

コンストラクタでCCharacterBaseを親クラスとして指定すると、 描画時に標準のビットマップとGCD、サウンド等が利用できます。
