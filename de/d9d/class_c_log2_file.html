詳解

テキストファイルへのログ保存クラス

system/log/ にテキストを保存します。 一定量のバッファを持ち、それがいっぱいにならないと出力しません。 なので、ログがファイルへ吐き出される前（メモリ上にあるうち）に いきなり落ちるとそのログは失われます。 重要なログを出力する場合はFlush()を呼ぶとバッファーされているログが 即座にファイルへ書き出されます。
