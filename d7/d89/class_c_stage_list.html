void CStageList::Initialize ( )

ディレクトリを検索し、リストを初期化

初期化

1) サブディレクトリ（都合上RINGということになっている）の一覧を生成 2) 各ディレクトリに関してステージを検索 2.5) 検索した各ステージディレクトリの正当性を検証 3) ステージが1個も見つからなかったリングはリストから削除
