説明

CDirectInput用イベント取得クラス

CDirectInputクラスがパッドの復帰処理を行うためにWM_ACTIVATEメッセージを拾う必要がある そのためにバックグラウンドタスクとして登録する 復帰の際にグローバルの g_input の RecoverDirectInput() をコールする

また、毎フレーム g_input の KeyLog() をコールしてキー状態ログを生成させる
